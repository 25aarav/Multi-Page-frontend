# Multi-Page Frontend Application

This project is a multi-page frontend application built with React, React Router, and Tailwind CSS. It consists of 4 main pages:

* Landing Page 1 (Homepage)
* Landing Page 2 (Additional content)
* Login/Signup Page
* Dashboard Page

## Project Setup

This project was bootstrapped with Vite.

## Dependencies

* React
* React DOM
* React Router DOM
* Lucide React (for icons)
* Tailwind CSS
